# Mapping Olist Brazilian Ecommerce Dependence

**Purpose**: 
This is a one-page interactive microsite built to serve as an executive summary and proof asset for the "Mapping Olist Brazilian Ecommerce Dependence" Power BI case study. It presents key diagnostic insights regarding seller concentration, category leadership, and marketplace activity over time to assess whether growth was broad-based or dependent on a narrow set of drivers.

## Preview Instructions
Since this is a static site without backend dependencies, you can preview it locally by:
1. Cloning or downloading the repository.
2. Opening `index.html` directly in any modern web browser (e.g., double-clicking the file).
3. Alternatively, use a local server like VS Code's "Live Server" extension for a smoother preview experience.

## Deployment
This project is configured to be compatible with GitHub Pages. To deploy:
1. Go to your repository settings on GitHub.
2. Navigate to "Pages" on the left sidebar.
3. Under "Build and deployment", select the `main` branch (or whichever branch contains your code).
4. Save, and your site will be live shortly.
